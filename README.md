# Purkka.css
Minimal, responsive and clean CSS framework for quick prototyping.
Made with SASS. Edit the sass variables to customize it for your use case!

## Elements
- navigation
- lists
- form elements
- buttons
- cards
- tabs
- footer

## Project Structure
./src
  - sass source files

./public
  - compiled purkka.css
  - index.html to view the css in action
